Два узла, обмен-ся м/с запросами и ответами. Клиент, сервер
Clients: psql(console), pgAdmin(web), idea ultimate etc.

Для подкл к серверу бд:
1. Хост - ip/домен имя компа с сервером бд. 127.0.0.1 или localhost (на то же устройство)
2. Порт - номер виртуального сетевого интерфейса, с которым ассоциирован сервер бд. по умолч сервер БД Пострге - 5432 порт
3. Имя пользователя - по умолч создается postgres с полным доступом (без ограничений)
4. Пароль польз - при установке
